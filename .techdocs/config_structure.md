# Config structure

```
{
  "modes": [
    {
      "name": "<mode_name>",
      "min_players": <mode_min_players>,
      "max_players": <mode_max_players>,
      "groups": [
        {
          "name": "<mode_group_name>",
          "min_players": <mode_group_min_player>,
          "max_players": <mode_group_max_player>
        }
      ]
    }
  ],
  "user_data": <arbitary_data>
}
```
